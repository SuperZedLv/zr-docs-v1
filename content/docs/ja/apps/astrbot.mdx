## BEST TOKEN 接続方法

AstrBot は、モデルプロバイダーとして BEST TOKEN への接続をサポートしており、ユーザーは BEST TOKEN を通じて様々な AI モデルサービスにアクセスし、利用することができます。

### 設定手順

#### BEST TOKEN API Key の取得

BEST TOKEN に登録してログインした後、上部ナビゲーションバーの「コンソール」をクリックし、「トークン管理」をクリックして、「トークンを追加」ボタンをクリックし、新しい API Key を作成し、適切な権限を選択してから「作成」をクリックします。

![APIキーの作成](/assets/astrbot/image.png)

作成が成功したら、「キーをコピー」ボタンをクリックして、生成された API Key をコピーします。

![APIキーのコピー](/assets/astrbot/image-1.png)

#### AstrBot での BEST TOKEN サービスプロバイダーの設定

AstrBot 管理パネルを開き、「モデルプロバイダー」ページに移動し、「新しいモデルプロバイダーを追加」ボタンをクリックします。

BEST TOKEN は OpenAI Chat Completion および Responses インターフェースを完全にサポートしています。「OpenAI」をクリックして、OpenAI プロバイダーの設定ページに進みます。

ポップアップダイアログで、API Base URL を BEST TOKEN のインターフェースアドレスに設定します。ご自身でデプロイした BEST TOKEN を使用するか、サービスプロバイダーが合法的なアップストリーム認証とコンプライアンス義務を持つ BEST TOKEN サービスであることを確認してください。BEST TOKEN をローカルにデプロイしている場合は、`http://localhost:3000/v1` のようなローカルアドレスを入力します。出所不明の API アドレスやキーを本番環境に接続しないでください。

次に、「API Key」フィールドに API Key を入力し、「保存」ボタンをクリックします。

![AstrBotプロバイダー設定](/assets/astrbot/image-2.png)

次に「保存」をクリックして、BEST TOKEN プロバイダーの設定を完了します。

#### サービスプロバイダーの適用

「プロファイル」ページに移動し、モデルセクションを見つけて、「デフォルトチャットモデル」を先ほど作成した BEST TOKEN プロバイダーに変更し、「保存」ボタンをクリックします。

![適用](/assets/astrbot/image-3.png)

これで、BEST TOKEN を AstrBot のモデルプロバイダーとして正常に設定できました。BEST TOKEN が提供する様々な AI モデルサービスに、AstrBot を通じてアクセスし、利用できるようになりました。
